# tech-notes Project Standards

This repository contains high-fidelity technical handbooks. All edits and new notes must strictly adhere to the project's standards.

## Mandatory Rules
All note content, structure, and formatting rules live in one canonical source, read it before any edit:
- **Project Standards**: @.agents/rules/standards.md

## Core Behavioral Mandate
- **Strict Explicit Modification**: Do not modify unassigned sections without instruction. However, when asked to review or optimize a note, proactive refactoring is permitted to align the text pedagogically, compact verbose phrasing, and correct logical inconsistencies in code examples according to the canonical standards.
