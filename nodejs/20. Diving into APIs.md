## JSON vs JavaScript Object

|Feature|JSON|JavaScript Object|
|---|---|---|
|**Keys**|Must be **strings** (in quotes)|Quotes **not required**|
|**Example**|`{"firstName": "Narendra"}`|`{firstName: "Narendra"}`|
|**Usage**|Data exchange format (APIs, files)|In-memory data in JS code|

---

## Accessing Request Body

- `req.body` contains the payload sent by the client
- You **can't directly** access `req.body` without middleware -- it will be `undefined`
- Must use **Express JSON middleware** to parse incoming JSON data

```js
// Add this BEFORE routes in your code
app.use(express.json());

app.post('/user', (req, res) => {
    console.log(req.body);          // now accessible
    console.log(req.body.firstName); // "Narendra"
    res.send("User received");
});
```

- `express.json()` is a built-in middleware that parses incoming requests with **JSON payloads**
