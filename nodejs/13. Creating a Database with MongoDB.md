## Setting Up MongoDB -- Two Ways

|Approach|Description|
|---|---|
|**Self-Managed (Local)**|Download MongoDB, install locally, manage backups/replicas/uptime yourself|
|**MongoDB Atlas (Recommended)**|MongoDB hosts the DB on cloud. Just access via connection string. No worries about uptime, backups, replicas|

---

## MongoDB Atlas Setup (Free Tier)

1. Visit **mongodb.com**
2. Click **"Try Atlas Free"**
3. Sign up (via Google or manual form)
4. Accept terms and submit
5. Complete a quick survey
6. **Cluster Setup**: Choose **M0 Free Tier** (512MB)
   - Select cloud provider: AWS, GCP, or Azure
   - Choose region (close to target users)
   - Click **Create Deployment**
7. **Create a DB User:** Set username/password
8. **Get Connection String**: Choose "Drivers"
   - Copy connection string for backend use
9. **Install MongoDB Compass** (GUI):
   - Paste connection URI and explore DB visually

## Connecting MongoDB with Node.js

### Using MongoDB Native Driver

```bash
npm install mongodb
```

```js
const { MongoClient } = require('mongodb');

const client = new MongoClient("mongodb+srv://<username>:<password>@cluster-url");

async function run() {
    await client.connect();
    const db = client.db("your-db-name");
    // CRUD operations here
}

run();
```

- Use `async/await` for CRUD operations
- Check MongoDB Native Driver docs for methods

### Basic CRUD Example

```js
async function run() {
    await client.connect();
    const db = client.db("your-db-name");
    const users = db.collection("users");

    // INSERT one document
    await users.insertOne({ name: "Akshay", age: 25, city: "Bangalore" });

    // INSERT multiple documents
    await users.insertMany([
        { name: "Ravi", age: 30, city: "Delhi" },
        { name: "Priya", age: 22, city: "Mumbai" }
    ]);

    // FIND all documents
    const allUsers = await users.find({}).toArray();
    console.log(allUsers);

    // FIND with filter
    const user = await users.findOne({ name: "Akshay" });
    console.log(user);

    await client.close();
}

run();
```

## Why Use Mongoose?

- **ODM** (Object Data Modeling) library for MongoDB and Node.js
- Features:
  - **Schema validation**: define structure for documents
  - **Middleware/hooks**: pre/post save, validate
  - **Model relationships**: references between collections
  - **Query helpers**: chainable query methods

```bash
npm install mongoose
```
