## Server Basics

- Node.js introduced the philosophy that **JavaScript can run outside the browser**, primarily on servers
- **What is a server?** A computer located somewhere else (remotely) that receives and responds to requests -- just like your own computer, but accessible over the internet
- **Running a website on a server** means running it on a computer (somebody's computer or on the cloud)
- **What is the cloud?** A collection of servers remotely located
- At the end of the day, a server is still a CPU (a machine) receiving the request. That receiver (that receiving computer) is known as a **server**: -

## V8 JavaScript Engine

- V8 is a JavaScript engine that follows the **ECMAScript standard**: **ECMAScript** = Standards / Rules for scripting languages (JavaScript, JScript, ActionScript)
- **70% of V8** is written in **C++**

### How Code Runs on a Machine

```
High-Level Language (JS, C++, etc.)
        |
        v
  Assembly Code
        |
        v
  Machine / Binary Code  (0s and 1s)
        |
        v
  CPU Processor (x86 / ARM)
```

- **V8 (written in C++)** compiles JS directly to machine code
- **x86**: instruction set used by Intel/AMD processors (desktops, servers)
- **ARM**: instruction set used by mobile devices, Apple Silicon (M1/M2/M3)
- Different processors understand different machine code -- compiled binaries are **platform-specific**: -

## Node.js on Top of V8

- V8's job is only to **execute JavaScript code** that follows core ECMAScript rules
- V8 has to follow **ECMAScript standards** and can't go beyond them

```
V8 Engine alone:
  Can    --> Execute JS code (ES standard)
  Can't  --> Access databases
  Can't  --> Make HTTP/network calls
  Can't  --> Read/write files from file system
```

- **Node.js gives extra "superpowers"** on top of V8 -- these are server-side APIs
- All the server-side capabilities come from **Node.js, not V8**: Node.js is a **C++ application with V8 embedded inside**: that's how JS runs outside the browser
- Node.js is **61% JS, 23% C++**

```
+------------------------------------------+
|              Node.js Runtime             |
|  +------------------------------------+  |
|  |     Server-Side APIs (Node.js)     |  |
|  |  (fs, http, net, crypto, etc.)     |  |
|  +------------------------------------+  |
|  |         V8 JavaScript Engine       |  |
|  |    (Executes JS per ES standard)   |  |
|  +------------------------------------+  |
+------------------------------------------+
```
